### ELasticsearсh

Данные подготовленны для заливки в эластик. 

Заливать можно программой `elasticdump` - устанавливать из NPM

`npm i -g elasticdump`

Залить это дело в эластик можно вот так

```
elasticdump --input=jsonline.json --output=http://localhost:9200/students --limit=10000
```

Поднять эластик на локале можно имея докер

```
docker-compose up -d
```
